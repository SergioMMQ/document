Documentos

Gestión de archivos y permisos en Odoo

Descripción

El módulo **Documentos** para Odoo 18 permite gestionar archivos, revisiones y permisos de edición de manera eficiente. Con una interfaz intuitiva, los usuarios pueden organizar documentos en carpetas, asignar permisos de acceso y realizar un seguimiento de las modificaciones realizadas.

Esta herramienta es ideal para equipos que necesitan colaborar y mantener un registro claro de sus documentos.
